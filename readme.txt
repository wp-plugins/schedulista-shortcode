== Installation ==

You can download and install the Schedulista plugin using the built in
WordPress plugin installer. If you download the Schedulista plugin manually,
make sure it is uploaded to "/wp-content/plugins/schedulista-shortcode/".

Activate Schedulista in the "Plugins" admin panel using the "Activate" link. 

After you install the widget, you can copy the shortcode with the parameters
for your business from the [web settings
page](https://www.schedulista.com/settings/web) and paste it into your
WordPress form.
